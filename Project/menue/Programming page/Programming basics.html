I. 'If' ladder

'If' statement
  • The if statement evaluates a condition (an expression that results in a boolean value).
  • If the condition is true, the code block inside the if statement is executed.
  • If the condition is false, the code block is skipped.
'Else' Statement:
  • The 'else' statement can be paired with an 'if' statement to provide an alternative code block that executes when the 'if' condition is false.
  • Only one of the code blocks (either 'if' or 'else') will execute based on the condition.
Syntax
  • if (condition) {
    // Code to execute if the condition is true
    } else {
    // Code to execute if the condition is false
    }
Example and explanation
  • Example
    • int number = 10;
    if (number > 5) {
    std::cout << "The number is greater than 5." << std::endl;
    } else {
    std::cout << "The number is 5 or less." << std::endl;
    }
  • Explanation
    • If number is greater than 5, the program prints "The number is greater than 5."
    Otherwise, it prints "The number is 5 or less."

I. 'Else if' ladder

Syntax
  • You can chain multiple conditions using else if for more complex decision-making.
    if (condition1) {
    // Code if condition1 is true
    } else if (condition2) {
    // Code if condition1 is false and condition2 is true
    } else {
    // Code if both condition1 and condition2 are false
    }
Example and explanation
  • Example
    • int number = 10;
    if (number > 10) {
    std::cout << "The number is greater than 10." << std::endl;
    } else if (number == 10) {
    std::cout << "The number is exactly 10." << std::endl;
    } else {
    std::cout << "The number is less than 10." << std::endl;
    }
  • Explanation
    • Here, the program checks if number is greater than 10, exactly 10, or less than 10, and executes the corresponding code block.

II. 'While' loop

Syntax
  • In C++, the while loop is a control structure used to repeatedly execute a block of code as long as a specified condition remains true. It is useful when you need to perform a task multiple times but do not know in advance how many iterations will be needed. while (condition) {
    // Code to execute as long as the condition is true
    }
