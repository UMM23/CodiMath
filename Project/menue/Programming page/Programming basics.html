Theory

I. 'If' ladder

'If' statement
  • The if statement evaluates a condition (an expression that results in a boolean value).
  • If the condition is true, the code block inside the if statement is executed.
  • If the condition is false, the code block is skipped.
'Else' Statement:
  • The 'else' statement can be paired with an 'if' statement to provide an alternative code block that executes when the 'if' condition is false.
  • Only one of the code blocks (either 'if' or 'else') will execute based on the condition.
Syntax
  • if (condition) {
    // Code to execute if the condition is true
    } else {
    // Code to execute if the condition is false
    }
Example and explanation
  • Example
    • int number = 10;
    if (number > 5) {
    std::cout << "The number is greater than 5." << std::endl;
    } else {
    std::cout << "The number is 5 or less." << std::endl;
    }
  • Explanation
    • If number is greater than 5, the program prints "The number is greater than 5."
    Otherwise, it prints "The number is 5 or less."

I. 'Else if' ladder

Syntax
  • You can chain multiple conditions using else if for more complex decision-making.
    if (condition1) {
    // Code if condition1 is true
    } else if (condition2) {
    // Code if condition1 is false and condition2 is true
    } else {
    // Code if both condition1 and condition2 are false
    }
Example and explanation
  • Example
    • int number = 10;
    if (number > 10) {
    std::cout << "The number is greater than 10." << std::endl;
    } else if (number == 10) {
    std::cout << "The number is exactly 10." << std::endl;
    } else {
    std::cout << "The number is less than 10." << std::endl;
    }
  • Explanation
    • Here, the program checks if number is greater than 10, exactly 10, or less than 10, and executes the corresponding code block.

II. 'While' loop

Syntax
  • In C++, the 'while' loop is a control structure used to repeatedly execute a block of code as long as a specified condition remains true. It is useful when you need to perform a task multiple times but do not know in advance how many iterations will be needed.

    while (condition) {
    // Code to execute as long as the condition is true
    }

Example and explanation
  • Example
    • int counter = 0;

    while (counter < 5) {
    std::cout << "Counter: " << counter << std::endl;
    counter++;
    }
  • Explanation
    • In this example, the loop will print the value of 'counter' from 0 to 4. If the initial value of 'counter' was 5 or more, the loop body would not execute at all.

II. 'Do...while' Loop

Syntax
  • The 'do...while' loop checks the condition after executing the loop body. This means the loop body will execute at least once, regardless of whether the condition is true initially.
    • do {
    // Code to execute
    } while (condition);
Example and explanation
  • Example:
    • int counter = 0;

    do {
    std::cout << "Counter: " << counter << std::endl;
    counter++;
    } while (counter < 5);
  • Explanation:
    • In this example, the loop will print the value of counter from 0 to 4. Even if the initial value of counter was 5 or more, the loop body would execute once before checking the condition.

II. Key Difference

Condition Check Timing
  • 'while' loop: The condition is checked before the loop body is executed.
  • 'do...while' loop: The condition is checked after the loop body is executed.
Execution Guarantee
  • 'while' loop: The loop body may not execute at all if the condition is false initially.
  • 'do...while' loop: The loop body is guaranteed to execute at least once.

III. 'For' Loop

Syntax
  • In C++, the 'for' loop is a control structure used to execute a block of code a specific number of times. It is particularly useful when the number of iterations is known beforehand. The 'for' loop provides a compact way to initialize, test, and increment a counter variable in a single line.

    • for (initialization; condition; increment) {
    // Code to execute
    }
Components
  • Initialization: This part is executed once at the beginning of the loop. It is typically used to initialize a counter variable.

  • Condition: This expression is evaluated before each iteration. If it evaluates to true, the loop body executes. If it evaluates to false, the loop terminates.

  • Increment: This part is executed after each iteration of the loop body. It is usually used to update the counter variable.
Example and explanation
  • Example:
    • for (int i = 0; i < 5; i++) {
    std::cout << "i: " << i << std::endl;
    }
    Explanation:
  • Initialization: 'int i = 0;' initializes the counter variable 'i' to 0.
  • Condition: 'i < 5' checks if 'i' is less than 5. If true, the loop body executes. If false, the loop terminates.
  • Increment: 'i++'' increments the counter variable 'i' by 1 after each iteration.

Nested 'for' Loops

Syntax
  • 'for' loops can be nested within each other to handle multi-dimensional data structures like matrices.
  • for (initialization1; condition1; increment1) {
    for (initialization2; condition2; increment2) {
    // Code to execute in the inner loop
    }
    // Code to execute in the outer loop
    }
Example and Explanation
  • #include

    int main() {
    int rows = 5;

    for (int i = 1; i <= rows; i++) { // Outer loop for rows
    for (int j = 1; j <= i; j++) { // Inner loop for columns
    std::cout << "* ";
    }
    std::cout << std::endl; // New line after each row
    }

    return 0;
    }
  • Explanation

    • Outer Loop:
  • 'for (int i = 1; i <= rows; i++)'': Iterates over each row from 1 to 'rows'.
    • Inner Loop:
  • 'for (int j = 1; j <= i; j++)': Iterates and prints '*'' for the number of times equal to the current row number.
    • Code Execution:
  • 'std::cout << "* ";': Prints a star followed by a space.
  • 'std::cout << std::endl;': Prints a new line after each row to format the output as a triangle.

Exercise 3: Reverse a Number Using do...while Loop (Answer and Explanation)

Answer:

#include

int main() {
int number, reverse = 0;
std::cout << "Enter an integer: ";
std::cin >> number;

int temp = number;
do {
int digit = temp % 10;
reverse = reverse * 10 + digit;
temp /= 10;
} while (temp != 0);

std::cout << "Reverse of " << number << " is: " << reverse << std::endl;
return 0;
}

Explanation:

  • The program prompts the user to enter an integer.
  • The do...while loop extracts each digit from the number, constructs the reversed number, and continues until all digits are processed.
  • The loop guarantees at least one execution even if the number is 0.

Exercise 5: Menu-Driven Program Using do...while Loop (Answer and Explanation)

Answer:

#include

int main() {
int choice;
double num1, num2, result;

do {
std::cout << "Menu:\n";
std::cout << "1. Addition\n";
std::cout << "2. Subtraction\n";
std::cout << "3. Multiplication\n";
std::cout << "4. Division\n";

if (choice >= 1 && choice <= 4) {
std::cout << "Enter two numbers: ";
std::cin >> num1 >> num2;
}

switch (choice) {
case 1:
result = num1 + num2;
std::cout << "Result: " << result << std::endl;
break;
case 2:
result = num1 - num2;
std::cout << "Result: " << result << std::endl;
break;
case 3:
result = num1 * num2;
std::cout << "Result: " << result << std::endl;
break;
case 4:
if (num2 != 0) {
result = num1 / num2;
std::cout << "Result: " << result << std::endl;
} else {
std::cout << "Error: Division by zero" << std::endl;
}
break;
case 5:
std::cout << "Exiting..." << std::endl;
break;
default:
std::cout << "Invalid choice. Try again." << std::endl;
break;
}
} while (choice != 5);

return 0;
}

Explanation:

  • The program displays a menu with options for arithmetic operations and exit.
  • The do...while loop ensures that the menu is displayed at least once and keeps prompting the user for a choice until they choose to exit.
  • The switch statement handles the different operations based on the user's choice.
  • For valid operations (choices 1 to 4), the program prompts the user to enter two numbers and performs the chosen operation.
  • If the user enters 5, the loop terminates, and the program exits.

Exercise 5: Print Fibonacci Series (Answer and Explanation)

Answer:

#include

int main() {
int n, t1 = 0, t2 = 1, nextTerm;
std::cout << "Enter the number of terms: ";
std::cin >> n;

std::cout << "Fibonacci Series: " << t1 << ", " << t2;

for (int i = 1; i <= n - 2; i++) {
nextTerm = t1 + t2;
std::cout << ", " << nextTerm;
t1 = t2;
t2 = nextTerm;
}
std::cout << std::endl;

return 0;
}

Explanation:

  • The program prompts the user to enter the number of terms in the Fibonacci series.
  • The first two terms (t1 and t2) are initialized to 0 and 1, respectively.
  • The for loop iterates to generate the next terms of the Fibonacci series, starting from the third term.
  • Each new term is the sum of the previous two terms, which are then updated for the next iteration.
  • The series is printed in a comma-separated format.
